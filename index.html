Cocogitto

Cocogitto is a CLI/automation toolbox for the conventional commit and semantic versioning specifications.

Quickstart

MIT licensed. Github repo

Verified commits

Create conventional-compliant commits with ease.

No dependencies

Cocogitto has two standalone binaries, the only system dependency is git.

Automatic changelog and version bumping

Automatically bump versions and generate changelogs with your own custom steps and workflows.

Conventional git log

Search your commit history matching conventional commit items such as scope and commit type.

Release profiles

Your branching model requires different steps for releases, prereleases, hotfixes? We've got you covered!

Conventional commits enforced via Github Actions

Check your commit compliance on every push to the remote and create releases from your CI pipeline.
